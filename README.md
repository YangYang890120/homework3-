```
#### 1.登入,登出
#### 2.會員及員工管理
#### 3.進貨,銷貨,存貨管理
#### 4.新增,修改,刪除
#### 5.關鍵字查詢,日期區間查詢
#### 6.列印報表
```
> [!NOTE]
> 在新增,修改進貨或出貨時,存貨也會同步進行動態更新
